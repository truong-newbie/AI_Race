## 1. Tổng quan

Assertion Detection xác định ngữ cảnh ngữ pháp của thực thể trong văn bản y khoa tiếng Việt. Hệ thống hỗ trợ 3 loại assertion:

| Assertion | Mô tả | Ví dụ |
|-----------|--------|--------|
| `isNegated` | Thực thể bị phủ định | "không ho", "chưa ghi nhận đau ngực" |
| `isHistorical` | Thực thể thuộc tiền sử | "tiền sử bệnh tim", "đã từng mắc" |
| `isFamily` | Thực thể thuộc tiền sử gia đình | "mẹ bệnh nhân có bệnh tim" |

Áp dụng cho 3 loại entity: `TRIỆU_CHỨNG`, `CHẨN_ĐOÁN`, `THUỐC`.

## 2. Kiến trúc

```
Input Text
    │
    ├── ClauseSegmenter ─────► segment_sentences() ──► List[SentenceScope]
    │                                        └──► _segment_clauses() ──► List[ClauseBoundary]
    │
    ├── find_cue_matches() ──► List[CueMatch] (negation, historical, family cues)
    │
    ├── resolve_entity_scope() ──► EntityScope
    │       ├── get_entity_sentence()
    │       ├── get_entity_clause()
    │       └── _find_relevant_cues()
    │
    ├── _is_cue_in_scope() ──► bool (áp dụng scope rules)
    │
    └── apply_scope_rules() ──► Dict[is_negated, is_historical, is_family]
            │
            └── RuleBasedDetector.detect() ──► EntityAssertion
```

### Luồng xử lý

1. **Segment text** thành sentences và clauses
2. **Tìm tất cả cues** (negation, historical, family) trong text
3. **Xác định sentence và clause** chứa entity
4. **Kiểm tra scope** cho từng cue:
   - Cue phải nằm trước entity
   - Không có contrast conjunction (nhưng, tuy nhiên) giữa cue và entity
   - Cue không vượt qua sentence boundary
5. **Áp dụng rule đặc biệt**: "tiền sử" sau family cue chỉ dùng isFamily

## 4. Scope Resolution

### 4.1 Scope Rules

| Rule | Mô tả |
|------|--------|
| Rule 1 | Cues không vượt qua sentence boundary (phân cách bởi `.`) |
| Rule 2 | Dừng tại semicolon khi phù hợp |
| Rule 3 | Conjunction handling: nhưng, tuy nhiên, song, hiện tại, còn |
| Rule 4 | Cues áp dụng cho entity lists: "không ho, sốt, khó thở" |
| Rule 5 | Conjunction exception: "không ho nhưng đau ngực" — chỉ ho bị negate |
| Rule 6 | "mẹ bệnh nhân từng mắc hen" → isFamily + isHistorical (từng = explicit historical) |
| Rule 7 | Entities trong "TIỀN SỬ" section → isHistorical |

### 4.2 Conjunction Patterns

```python
CONJUNCTION_PATTERNS = [
    # Contrast (nhưng, tuy nhiên, song)
    (re.compile(r"nhưng\b"), "contrast"),
    (re.compile(r"tuy\s+nhiên\b"), "contrast"),
    (re.compile(r"song\b"), "contrast"),

    # Concurrent (còn, hiện tại)
    (re.compile(r"còn\b"), "concurrent"),
    (re.compile(r"hiện\s+tại\b"), "concurrent"),
]
```

### 4.3 Key Function: `_is_cue_in_scope`

```python
def _is_cue_in_scope(
    text: str,
    cue: CueMatch,
    entity_start: int,
    sentence_scope: SentenceScope,
    clause_scope: ClauseBoundary
) -> bool:
    # 1. Cue must be before entity in same clause
    if cue.end <= entity_start and cue.end > clause_scope.start:
        return True

    # 2. Check conjunction between cue and entity
    clause_text = text[clause_scope.start:clause_scope.end]
    for pattern, conj_type in CONJUNCTION_PATTERNS:
        conj_match = pattern.search(clause_text)
        if conj_match:
            conj_pos = clause_scope.start + conj_match.start()
            # If contrast conjunction between cue and entity → cue out of scope
            if cue.end <= conj_pos < entity_start and conj_type == "contrast":
                return False

    return False
```

### 5.2 Confidence Scoring

```python
def calculate_confidence(cue_priority: int) -> float:
    if cue_priority > 0:
        return min(0.7 + (cue_priority / 100), 0.99)
    return 0.8  # default
```

| Priority | Confidence | Ví dụ |
|----------|------------|--------|
| 30 | 1.0 | loại trừ, tiền sử gia đình |
| 25 | 0.95 | không ghi nhận, âm tính với, đã sử dụng |
| 20 | 0.90 | không có, không thấy, tiền sử, đã từng |
| 15 | 0.85 | không, chưa, từng, gia đình |

## 6. API và cách sử dụng

### 6.1 Basic Usage

```python
from src.assertion.rules import RuleBasedDetector

detector = RuleBasedDetector()

text = "Bệnh nhân không ho, mẹ có tiền sử bệnh tim."

# Detect for single entity
result = detector.detect(text, start=14, end=16, entity_type="TRIỆU_CHỨNG")

print(result.status.is_negated)   # True
print(result.status.is_historical) # False
print(result.status.is_family)     # False
print(result.status.confidence)     # 0.90
```

### 6.2 Batch Detection

```python
entities = [
    {"start": 14, "end": 16, "type": "TRIỆU_CHỨNG"},
    {"start": 28, "end": 36, "type": "CHẨN_ĐOÁN"},
]
results = detector.detect_all(text, entities)
```

### 6.3 Output Formats

```python
# Dictionary format
result.to_dict()
# {'entity_text': 'ho', 'is_negated': True, 'is_historical': False, ...}

# List format (for competition output)
result.to_list()
# ['isNegated']
```

### 6.4 Get All Cues in Text

```python
cues = detector.get_cues(text)
for cue in cues:
    print(f"{cue.text} [{cue.start}:{cue.end}] - {cue.cue_type.value}")
```
