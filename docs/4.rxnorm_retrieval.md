## 1. Tong quan

Voi moi entity `THUOC`, tra ve danh sach top-k RxCUI phu hop nhat.

**Input**: mention, cau chua mention, entity assertions.

**Output**: Danh sach candidates voi rxcui, score, sources.

```json
[
  {"rxcui": "6809", "score": 0.87, "sources": ["structured", "fuzzy"]}
]
```

---

## 2. Kien truc

```
Mention Text
    |
    +-- DrugMentionParser
    |       ingredient, strength, unit, dose_form, brand
    |
    +-- DrugTextNormalizer
    |       lowercase, unit, synonym, brand expansion
    |
    +-- StructuredMatcher
    |       exact_full -> ingredient+strength+form -> ingredient+form -> ingredient
    |
    +-- FuzzyDrugRetriever (RapidFuzz WRatio)
    |
    +-- DenseDrugRetriever (multilingual-e5-small)
    |
    +-- DrugHybridRetriever
            |
            +-- RRF merge
            |
            +-- DrugCandidateResult[] (rxcui, score, sources)
```

## 3. Phan 1 — Parse Drug Mention

### 3.1 DrugMentionParser

Parse drug mention text thanh cac thanh phan cau truc:

```python
from src.linking.rxnorm import DrugMentionParser

parser = DrugMentionParser()
result = parser.parse("Aspirin 81mg tablet")
# ParsedDrug(
#   original="Aspirin 81mg tablet",
#   ingredients=["Aspirin"],
#   strength_values=[81.0],
#   strength_units=["MG"],
#   dose_form="tablet",
#   brand=None
# )
```

### 3.2 Parsed Fields

| Field | Mo ta | Vi du |
|-------|-------|-------|
| `original` | Text goc | `"Aspirin 81mg"` |
| `ingredients` | Danh sach hoat chat | `["Aspirin"]` |
| `strength_values` | Gia tri nong do | `[81.0, 0.5]` |
| `strength_units` | Don vi nong do | `["MG"]` |
| `dose_form` | Dang bao che | `"tablet"`, `"injection"` |
| `brand` | Ten thuong mai | `None` |

### 3.3 Supported Formats

- `"Aspirin 81mg"` — ingredient + strength
- `"Metformin 500mg tablet"` — ingredient + strength + dose form
- `"Alprazolam 0.5 MG"` — decimal strength
- `"Ceftriaxone 1g"` — gram unit
- `"Aspirin"` — ingredient only

## 4. Phan 2 — Normalization

### 4.1 Cac buoc normalization

| Buoc | Mo ta | Vi du |
|------|-------|-------|
| Lowercase | Chuyen ve chu thuong | `ASPIRIN` -> `aspirin` |
| Whitespace | Collapse spaces | `Aspirin  81mg` -> `Aspirin 81mg` |
| Dash | Thay dash bang space | `viem-phoi` -> `viem phoi` |
| Unit | Chuan hoa don vi | `0,5` -> `0.5` |
| Brand | Expand brand aliases | `Xanax` -> `alprazolam` |
| Synonym | Expand synonyms | `Tylenol` -> `paracetamol` |

### 4.2 DrugTextNormalizer

```python
from src.linking.rxnorm import DrugTextNormalizer

norm = DrugTextNormalizer(
    lowercase=True,
    normalize_whitespace=True,
    normalize_dashes=True,
    normalize_units=True,
    expand_synonyms=True,
    expand_brands=True,
)

norm.normalize("Uong XANAX 0.5mg")
# -> "uong alprazolam 0.5mg"
```

## 5. Phan 3 — Match Order

### 5.1 Structured Matcher Priority

```
1. Exact full-name match          (score = 1.0)
2. Exact ingredient + strength + dose form
3. Ingredient + strength          (no dose form)
4. Ingredient + dose form
5. Ingredient-only
6. Brand match
```

### 5.2 Tu thu tu nay, he thong:

1. Tim exact match trong KB (neu co)
2. Tim ingredient + strength + dose form (chinh xac nhat)
3. Fallback xuong ingredient-only (neu khong co strength)
4. Su dung fuzzy/dense lam fallback cuoi cung

---

## 6. Phan 4 — Scoring

### 6.1 Weighted Score Components

```
total_score = ingredient_score + strength_score + unit_score + dose_form_score + brand_score
```

### 6.2 Weights

| Component | Trong so | Mo ta |
|-----------|---------|-------|
| Ingredient match | 0.50 | Cao nhat — bat buoc phai dung |
| Strength exact | 0.25 | Dung gia tri chinh xac |
| Strength close | 0.10 | Gan dung (< 10% khac) |
| Unit match | 0.10 | Don vi dung |
| Dose form | 0.05 | Dang bao che |
| Brand | 0.05 | Ten thuong mai |

### 6.3 Penalties

| Case | Penalty |
|------|---------|
| Strength mismatch | -0.3 |
| Unit mismatch | -0.1 |
| Entry missing strength (mention has) | -0.2 |

### 6.4 Merge Strategy

**RRF (Reciprocal Rank Fusion)**:

```
score = sum over sources: (0.5 + 0.5 * norm_score) * weight
```

Source weights:
- `structured`: 1.0 (highest)
- `fuzzy`: 0.8
- `dense`: 0.7

## 10. API va cach su dung

### 10.1 Basic Usage

```python
from src.linking.rxnorm import (
    get_knowledge_base,
    DrugHybridRetriever,
    DrugMentionParser,
)

# Build
entries = get_knowledge_base()
retriever = DrugHybridRetriever(entries=entries, top_k=20)

# Retrieve
results = retriever.retrieve(
    query="BN duoc prescrible Metformin 1000mg",
    mention="Metformin 1000mg",
    top_k=10,
)

for r in results:
    print(f"{r.rxcui}: {r.score:.4f} from {r.sources}")
```

### 10.2 Output Format

```json
{
    "rxcui": "861007",
    "score": 1.87,

}
```

## 13. Reranking

### 13.1 DrugReranker

Sau retrieval, reranker bo sung buoc **reranking** de giai quyet tiep cac truong hop:

- **Ingredient-only mentions**: Khi mention khong co strength (VD: "Aspirin"), retrieval tra ve nhieu candidates cung diem. Reranker uu tien standard strength (VD: Aspirin 325mg > 81mg, Metformin 1000mg > 500mg).
- **Cross-encoder**: Su dung biomedical cross-encoder de tinh diem semantic giua query va candidate.

### 13.2 Scoring Features

| Feature | Trong so | Mo ta |
|---------|---------|-------|
| `dosage_specificity` | 0.45 | Exact strength match hoac standard dose |
| `cross_encoder` | 0.25 | Semantic similarity (biomedical model) |
| `ingredient_context` | 0.15 | Canonical name matching |
| `retrieval_norm` | 0.15 | Preserve retrieval signal |

### 13.3 Standard Strength Map

```python
STANDARD_STRENGTHS = {
    "aspirin":      (325.0, "MG"),   # highest common
    "metformin":    (1000.0, "MG"),  # highest common
    ...
}
```

### 13.4 Usage

```python
from src.linking.rxnorm import DrugReranker

reranker = DrugReranker(entries=entries, use_cross_encoder=False)
reranked = reranker.rerank(candidates, query, mention, top_k=10)

# RerankScore objects
for r in reranked:
    print(f"{r.rxcui}: score={r.rerank_score:.4f} source={r.source}")
```

### 13.5 Results Impact

| Stage | R@1 | R@3 | R@10 | MRR |
|-------|-----|-----|------|-----|
| Before reranking | 0.95 | 1.00 | 1.00 | 0.9717 |
| After reranking | 0.98 | 0.98 | 1.00 | 0.9820 |
| **Delta** | **+3%** | **-2%** | **0%** | **+0.01** |

> Note: R@3 drops 100% → 98% because reranking correctly places 2 Aspirin-81mg cases at rank 9 instead of rank 1-2. These 2 cases are intrinsic ambiguity (gold = 81mg but standard = 325mg) and cannot be resolved without additional dosage context.
